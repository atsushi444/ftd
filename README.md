<h3 id="g-g">G, g</h3>
<ul>
<li>
<p>发浊辅音 “g”，注意 ge 和 gi 被 gue 和 gui 替换了</p>
</li>
<li>
<p>ge 和 gi 发 je 和 ji 的音，也就是 /he/ 和 /hi/。</p>
</li>
</ul>
<p>-ge = je ; ege = eje <br>
gi = ji ; igi = iji</p>
<ul>
<li>分音符：güi；güe，u和i都要发出来，换句话说就是一个辅音+两个元音<br>
而 gui 和 gue 则只发 /gi/ 和 /ge/ 的音，换句话说就是一个辅音+一个元音</li>
</ul>
